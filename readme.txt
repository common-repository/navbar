== Description ==

Navbar is a WordPress admin plugin which adds a handy navigation bar to organize your WordPress blog shortcuts.

<strong>Main Features:</strong>
<ul>
	<li>Quickly access your favorite pages or administration options</li>
	<li>Display handy information like latest comments and visitor</li>
	<li>Have 3rd part expandability through 'navbar_info' filter (for developers)</li>
	<li>Easily remove it from the screen anytime using a hotkey (default Ctrl - e)</li>
</ul>

You can use Navbar to add shortcuts to your plugin options, comments or other frequently accessed WordPress screens or posts on your blog.

Especially interesting is 3rd party plugin integration as more information can be added to the Navbar info area. On one of the screenshots you can see integration with <a href="http://wordpress.org/extend/plugins/plugin-central/">Plugin Central</a> where it displays a link to automatically upgrade all of your outdated plugins at once.

== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Go to the Plugins page and activate the plugin.
3. Visit plugin option and adjust as you wish
4. Navbar will follow you anywhere you go!
